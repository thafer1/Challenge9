  ## Installation 
  1. Clone repo onto local storage
  2. Navigate to folder
  3. Run npm init -y and npm install inquirer

  ## Usage 
  1. Run node index.js
  2. Enter info as prompted
  3. Enjoy your README!<br />
  [Application demonstration](https://youtu.be/wVgWObArOJA)
